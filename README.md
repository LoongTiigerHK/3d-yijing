# 3d-yijing
易经全息有向矢量分析系统 (I-Ching 3D Vector Analysis)
本系统是一个基于 3D 矢量空间建模 的易经分析工具。它将传统的 64 卦阴阳爻结构映射至三维欧几里得空间，通过 9 种能量变迁算子（Operators），定量化地展示卦象之间的能量位移与势能变化。
🌌 系统核心理论1. 空间映射模型系统将每一卦的三个主爻（初、二、三爻）视为三维坐标系中的基向量：阴爻 (⚋) 映射为坐标数值 2阳爻 (⚊) 映射为坐标数值 1因此，任意一单卦 $G$ 可表示为空间坐标点 $P(x, y, z)$，其中 $x, y, z \in \{1, 2\}$。2. 能量矢量定义我们将“下卦”定义为 源 (Source)，代表内场势能；将“上卦”定义为 汇 (Sink)，代表外场表达。基础能量矢量 $\vec{V}_{base}$ 定义为从源点到汇点的有向线段：$$\vec{V}_{base} = P_{upper} - P_{lower}$$其 模长 (Magnitude) $|\vec{V}_{base}|$ 代表了该卦象在平衡态下的基准能量强度。3. 变迁算子 $L_{1 \dots 9}$系统引入了 9 种激发态算子，模拟阴阳爻变动时能量场的扭曲与跃迁。算子通过改变源点与汇点的局部坐标，计算出新的矢量场分布，揭示卦象在动态变化中的能量增量 $\Delta M$。
🛠️ 功能特性全量覆盖：完整补全 64 卦名称及 Unicode 符号（如 ☵☲ 水火既济）。实时交互：通过下拉菜单实时切换卦象，动态更新 3D 矢量视图。量化数据：侧边图例实时显示各算子的 激发模长，支持能量状态（🟢 增强 / 🔴 减弱 / ⚪ 平衡）的定量分析。深色学术视觉：采用高对比度霓虹色系，强制深色模式，便于观察复杂的空间几何关系
